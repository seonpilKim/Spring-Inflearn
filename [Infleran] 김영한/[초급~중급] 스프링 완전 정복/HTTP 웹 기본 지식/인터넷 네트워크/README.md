## PORT
- 한 번에 둘 이상 연결해야 하면?
    - ![](imgs/15.PNG)
    - TCP or UDP 패킷에 출발지 PORT, 도착지 PORT 정보를 추가하여 전송한다.
- PORT는 같은 IP 내에서 프로세스를 구분하는 용도로 쓰인다.
    - ![](imgs/16.PNG)
- 0 ~ 65535 할당 가능
- 0 ~ 1023 : 잘 알려진 포트이므로 사용하지 않는 것이 좋다
    - FTP - 20, 21
    - TELNET - 23
    - HTTP - 80
    - HTTPS - 443

    - IP를 갱신할 때 마다, 새로운 IP를 클라이언트가 알고 있어야 한다.
- DNS(Domain Name System)
    - 전화번호부
    - 도메인 명을 IP주소로 변환
